LambdaTest configuration options.

outOk

output lines for asserts that succeed.

outHeader

output an initial header line.

outSummary

output an final summery line.

outExceptionTrace

output a stack trace of any unexpected exceptions.

onlyIfFail

output only those parts that contain failures.

useColor

use colors in the output.

indent

number of characters to indent each level of output.

includeTags

actions only included if they include at least one of these tags or includeTags is empty

excludeTags

actions only include if they contain none of these tags

includeAction

function to decide if an action is included based on tags options and the action's tags
